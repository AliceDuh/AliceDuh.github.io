# Alice Duhaut — GitHub Pages Website Template

A simple, responsive academic website template built from the uploaded CV.

## Files

- `index.html` — main website
- `styles.css` — styling
- `CV-2.pdf` — CV linked from the site

## How to publish on GitHub Pages

1. Create a repository named `your-username.github.io`.
2. Upload `index.html`, `styles.css`, and `CV-2.pdf` to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`, then save.
6. Your site will appear at `https://your-username.github.io/`.

## Things to customize

- Replace `#` links in the research cards with real paper, journal, slide, and code links.
- Add a headshot by placing an image such as `headshot.jpg` in the folder and editing the hero section.
- Update colors in `styles.css` under the `:root` block.
